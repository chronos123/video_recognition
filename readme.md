### 项目结构

#### 后台运行：

注意从requirements.txt中安装包

back end的seetaface文件夹下 使用cmd运行python app.py启动后台；

#### 前台运行：

下载node-js：[https://nodejs.org/en/download/]选windows版本

安装之后打开命令行，在dist文件夹下打开cmd，输入npm install http-server -g

之后输入http-server

然后在浏览器中打开对应的127.0.0.1:8081网页

保证后台已运行的情况下点击开始识别
